# Air Canvas CV

This project's goal is to get familiar with OpenCV C++ library in a fun and simple way.

## Project structure

```text
air_canvas_cv/
├── CMakeLists.txt          # Project build file
├── README.md
├── .gitignore
├── src/
│   ├── main.cpp            # Main loop
│   ├── AirCanvas.cpp       # Main features class
│   └── AirCanvas.h
└── build/                  # Build output
```
